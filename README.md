# Theology

A repository of theological writings — short, plain-language answers
to questions about the Christian faith.

## How It Is Written

Every document in this repository is written by hand through study,
research, and prayer. AI is then used to edit and enhance the
writing, helping with clarity, structure, and consistency.

Every change and every commit is reviewed by a human first —
human-in-the-loop, or HITL. Nothing lands in this repository without
that review.

## What You Will Find

- Q&A documents that answer common questions about the Bible, God,
  and the Christian life.
- Plain-language explanations written so a middle school student can
  read and understand them.
- A friendly, loving tone — encouraging, never shaming.

## Doctrinal Anchor

The writings are anchored in Reformed theology. Two sources are used
again and again:

- **Westminster Confession of Faith (WCF)** for doctrine, linked to
  the Orthodox Presbyterian Church version at
  <https://www.opc.org/wcf.html>.
- **English Standard Version (ESV)** for Scripture, linked through
  <https://www.esv.org/>.

## Layout

- `Next/` — current working set of documents.
- `Next/CLAUDE.md` — style and formatting guide for new documents.

## A Note on AI

AI helps with phrasing, formatting, and consistency. It does not
decide doctrine. Every claim about Scripture or Reformed teaching is
checked by a human before it is committed.
